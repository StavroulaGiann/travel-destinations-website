# Travel Destinations Website

Multi-page static website showcasing different travel destinations.
Each destination has its own dedicated page with structured content
and images.

## Features
- Multi-page website structure
- Dedicated pages for each travel destination
- Consistent layout and styling across all pages
- Use of images and icons
- Clean and semantic HTML structure

## Pages
- Home page
- Destination pages (e.g. Ioannina, Pilio, Vienna, Budapest, etc.)

## Technologies
- HTML5
- CSS3

## How to View
Open the `index.html` file in any modern web browser and navigate
through the destination pages using the available links.
